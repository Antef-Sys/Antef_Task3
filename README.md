# 📁 Antef_Task3

## 📌 Project Overview

This project demonstrates the use of **HTML and CSS selectors** including:

* Element selectors
* Class selectors
* ID selectors
* Pseudo-selectors (`:nth-child`)
* Gradient backgrounds
* Styling specific text using `<span>`

---

## 🏗️ Structure

### 🔹 HTML

The HTML file includes:

* **Header section**

  * Multiple headings (`h1`, `h2`, `h4`)
  * Use of class `.head`
  * Highlighted word using `<span>`

* **Main section**

  * Paragraphs with and without `id` and `class`
  * Highlighting specific words using `<span>`
  * Unordered list (`ul`) with links
  * Ordered list (`ol`) with styled items

---

### 🎨 CSS Features

#### ✅ Background Gradient

* Applied gradient to the entire page using:

```
linear-gradient(90deg, #b45d1b 0%, #daae51 100%)
```

#### ✅ Text Styling

* `.head` → styles all headings
* `.highlight` → styles specific word inside heading
* `.phighlight` → styles word inside paragraph

#### ✅ ID Selectors

* `#para` → custom paragraph styling
* `#bglime` → gradient background for heading
* `#two`, `#three` → specific list item colors

#### ✅ Element Selectors

* `p` → styles all paragraphs
* `ul li` → styles list items

#### ✅ Pseudo Selectors

* `ul li:nth-child(1)` → first link background
* `ul li:nth-child(2)` → second link background
